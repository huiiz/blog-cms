# 博客搭建完整教程

本文详细介绍如何使用免费工具搭建一个功能完善的个人博客系统。

## 工具选择

### 为什么选择这套技术栈？

1. **Astro** - 性能出色的静态站点生成器
2. **TinaCMS** - 强大的 Git-based CMS
3. **Cloudflare Pages** - 快速可靠的免费托管
4. **GitHub** - 代码仓库和 CI/CD

## 部署步骤

### 第一步：准备 GitHub 仓库

1. 创建新的 GitHub 仓库
2. 克隆到本地
3. 推送代码

### 第二步：连接 Cloudflare Pages

1. 登录 Cloudflare Dashboard
2. 选择 Pages
3. 连接到 GitHub 仓库
4. 配置构建命令

### 第三步：配置 TinaCMS

1. 在 GitHub 创建 GitHub App
2. 获取 Client ID 和 Secret
3. 配置环境变量

## 常见问题

### Q: 需要多少钱？

**A: 完全免费！** 这套方案不需要任何费用。

### Q: 需要编程基础吗？

**A: 基本不需要。** TinaCMS 提供了可视化编辑界面。

## 总结

按照本文的步骤，你可以在 30 分钟内搭建一个专业级的个人博客。开始吧！
