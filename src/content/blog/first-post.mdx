# 欢迎来到博客

这是一个使用 Astro + TinaCMS 构建的现代化博客系统。

## 主要特性

- **静态生成** - 超快的加载速度和完美的 SEO
- **在线编辑** - 通过 TinaCMS 可以直接在浏览器中编辑内容
- **免费托管** - 使用 Cloudflare Pages 完全免费托管
- **精美主题** - 支持暗黑模式，响应式设计

## 快速开始

### 1. 创建新文章

在管理后台点击"新建文章"，系统会自动在 GitHub 上创建新文件。

### 2. 编辑内容

点击右上角的 Tina 图标进入可视化编辑模式，修改文章内容。

### 3. 发布更新

保存后系统会自动创建 Pull Request，合并后网站会自动更新。

## 总结

这个博客系统简单、强大、免费，非常适合个人博主使用。快来试试吧！
