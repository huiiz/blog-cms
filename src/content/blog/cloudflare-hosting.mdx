# Cloudflare 免费托管指南

Cloudflare 为开发者提供了非常慷慨的免费额度，本文详细介绍如何利用这些服务。

## Cloudflare Pages

### 免费额度

- 无限带宽
- 500 次构建/月
- 100 个站点
- 自定义域名
- 免费 SSL

### 支持的框架

- Astro
- Next.js
- Gatsby
- Hugo
- 以及所有静态站点

## Cloudflare Workers

### 免费额度

- 100,000 请求/天
- 10ms CPU 时间/请求
- 30 个 Worker 脚本

### 适用场景

- API 代理
- 图像处理
- 重定向服务
- A/B 测试

## D1 数据库

### 免费额度

- 5GB 存储
- 无限数据库
- 无限查询

## R2 对象存储

### 免费额度

- 10GB 存储
- 100 万次 A 类操作
- 1000 万次 B 类操作

## 最佳实践

1. 合理利用边缘计算
2. 优化资源使用
3. 监控用量
4. 及时清理不需要的资源

## 总结

Cloudflare 的免费额度对于个人项目来说非常充足，值得充分利用！
